Netflix--data-analysis-excel
🎬 Netflix Content Analysis — Excel Project

 📌 Overview
Analysis of 8,800+ Netflix titles to uncover content trends,
popular genres, growth patterns and country distributions.

🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Charts, Formulas, Dashboard)

📊 Key Findings
- Netflix has 6,131 Movies (70%) and 2,676 TV Shows (30%)
- International Movies is the most common genre
- Content additions grew rapidly from 2015 to 2019
- USA produces the most Netflix content globally
- TV-MA is the most common rating (adult content)
- Average movie runtime is 99.5 minutes

📁 File Structure
| Sheet | Description |
|-------|-------------|
| 1_Raw_Data | Original dataset |
| 2_Cleaned_Data | Cleaned & processed data |
| 3_Summary | KPIs and key metrics |
| 4_Pivot_Tables | Data aggregations |
| 5_Charts | Visual analysis |
| 6_Dashboard | Final insights dashboard |
